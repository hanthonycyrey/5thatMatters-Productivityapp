# 5 That Matter 🐸

A brutally minimal task management app for people who know that focus is a superpower.

## Philosophy

Most task managers are feature-bloated distractions dressed up as productivity tools. **5 That Matter** strips away everything except what actually matters: *doing the most important work first*.

The app is built on four proven frameworks:
- **Eisenhower Matrix** - Distinguish important from urgent
- **Buffett's 25/5 Rule** - Cap your focus to 5 tasks maximum
- **80/20 Principle** - Identify which tasks drive 80% of results
- **Eat That Frog** - Do the hardest task first

Result: A task manager that enforces *actual focus* instead of just promising it.

## Core Constraints

✋ **Maximum 5 active tasks** - No exceptions. No cheating.  
📌 **Only "Important" tasks make the list** - Urgent-but-trivial goes to Delegate.  
🐸 **The Frog always leads** - Hardest/most impactful task shown first.  
🎯 **80/20 flagging** - Mark tasks that drive majority of results.  
🚫 **No multicolored tags, no drag-and-drop theater** - Just work.  

## How It Works

### Adding a Task

You have an idea. Before it enters your 5:

1. **Type the task**
2. **Modal appears:** "Is this Important?" (Yes/No)
3. **Follow-up:** "Is this Urgent?" (Yes/No)

Your answers determine where it lands:

| Important | Urgent | Destination |
|-----------|--------|-------------|
| ✓ | ✓ | Active 5 (or Waiting Room if full) |
| ✓ | ✗ | Active 5 (scheduled lower) |
| ✗ | ✓ | Delegate pile (not your job) |
| ✗ | ✗ | Graveyard (with gentle nudge: "Drop this.") |
